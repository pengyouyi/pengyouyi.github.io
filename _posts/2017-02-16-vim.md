# vim简单使用

## 新建一个文本文件new
`vim test.txt`

例如：新建一个.babelrc 文件
```shell
vim .babelrc
```

## 编辑文件edit
```shell
i
```

## 保存并退出文件save
先按 ESC 键退出，然后 ` :wq `
