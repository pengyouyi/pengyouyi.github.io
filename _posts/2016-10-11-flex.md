# flex布局

# demo
```html
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>flex布局example</title>
</head>
<style>
*{
	margin:0;
	padding: 0;
}
#wrap{
  	display: flex;
  	//justify-content: space-between;
  	//justify-content: flex-end;
  	//align-items:center;
  	flex-direction: column;
  	margin:0 auto;
  	width:1000px;
  	height: 300px;
  	border:1px solid red;
}
header,footer{
	height:40px;
	background: #f60;
}
.main{
	display: flex;
	height:160px;
	margin:20px 0;
}
.list{
	margin-right:20px;
	background: yellow;
}
.li1{
	width:100px;
}
.li2{
	flex:1;
}
.main #l3{
	width:40px;
	margin-right:0;
}
</style>
<body>
<div id="wrap">
	<header>头部</header>
	<div class="main">
		<div class="list li1">1</div>
		<div class="list li2">2</div>
		<div class="list" id="l3">3</div>
	</div>
	<footer>脚部</footer>
</div>
	
	
	</script>
</body>
</html>
```
